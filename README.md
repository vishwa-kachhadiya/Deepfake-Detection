# Deepfake Detection: Demodulate Synthetic Videos Using Deep Learning Models

Official implementation of our research paper published in **IEEE INDIACom 2025**.

## 📖 Overview

Deepfake technology has advanced rapidly, making it increasingly difficult to distinguish manipulated videos from authentic ones. This project presents a deep learning-based framework for detecting synthetic videos by learning discriminative visual features from manipulated facial content.

The proposed approach aims to improve the reliability and accuracy of deepfake detection and can be applied in digital forensics, social media verification, and misinformation detection.

---

## ✨ Features

- Deepfake video detection using Deep Learning
- Video preprocessing and frame extraction
- Face detection and feature extraction
- Deep learning model training
- Model evaluation and performance analysis
- Visualization of prediction results
